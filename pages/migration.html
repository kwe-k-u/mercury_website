How to grant Mercury access to your Jumia Account

Migration Guide

If you are using the mobile process, Check the Mobile migration process
  1. Enter your email. It can be different from your Jumia email
  2. Enter a strong password and retype it to confirm

  1. Log into your Jumia account via Jumia's seller center website
  2. From the Settings Menu, select Manage Users
  3. Click on 'Add User' and fill the fields as shown below

    4. Role: Full Seller Access

    Email address: mercury@duala.com

    Name: Mercury

    Default Language: English

  4. Save the new account

  1. In the email field on this page, enter the email you used for the Mercury registration. This can be different from your jumia email
  2. Copy the API key text associated with the newly added user(Mercury)
  3. Paste the copied API key into the API key form on the right of this page
  4. Click on submit

  2. Wait for the app to confrim the migration was successful
